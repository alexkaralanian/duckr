# Duckr

Duckr is a social media Twitter clone built with React-Redux. (from TylerMcGinnis.com's Redux tutorial )

Features:

* React 15.4
* Redux
* React Router v4
* Webpack 2
* CSS Modules
* Hot Module Reloading

Coming Soon:

* React-Redux
* Firebase deployment
* NoSQL DB
* Facebook OAuth login
* ImmutableJS
